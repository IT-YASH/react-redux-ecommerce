# React Redux E-Commerce App

## Tech Stack
- React
- Redux Toolkit
- React Router
- Tailwind CSS
- DummyJSON API

## Features
- User authentication with JWT
- Product listing with category filter
- Pagination
- Add to cart using Redux
- Checkout page with total price
- Responsive UI using Tailwind CSS

## How to Run Locally
```bash
npm install
npm run dev
